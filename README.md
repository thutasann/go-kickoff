# GoLang Kick off

This is the repo where I start digging into GoLang.

## Available Scripts

### Install Go

Install `Go` in your machine [Here](https://go.dev/doc/install)

### Running `Go` files

> Uncomment codes and run command :

```bash
go run main.go
```

### Build `Go` files

> Uncomment codes and run command :

```bash
go build main.go
```

### Format `Go` files

```bash
go fmt main.go
```
