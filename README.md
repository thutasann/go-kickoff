# GoLang Kick off 

This is the repo where I start digging into GoLang.

## Available Scripts

### Install Go

Install `Go` in your machine [Here](https://go.dev/doc/install)


### Change dir

```
cd src
```

### Run Go

```bash
go run main.go
```

### Build Go

```bash
go build main.go
```

### Format Go

```bash
go fmt main.go
```
